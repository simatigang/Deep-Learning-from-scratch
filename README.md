## 📌 학습 목표
외부 라이브러리(PyTorch 등)에 의존하지 않고, **Python과 Numpy만으로** 신경망의 핵심 알고리즘을 직접 구현하며 딥러닝의 하부 구조를 이해함.

---

## 🛠 주요 학습 내용 (핵심 요약)

### 1. 퍼셉트론 & 신경망 기초 (Ch 1~3)
- **활성화 함수(Activation Function):** Step, Sigmoid, ReLU 함수의 특성 비교.
- **회로 관점의 고찰:** 논리 회로(AND, NAND, OR)를 퍼셉트론으로 구현하며, 단층 퍼셉트론의 한계(XOR 문제)와 다층 구조의 필요성을 체감함.

### 2. 오차역전파법 (Backpropagation, Ch 5) ⭐
- **계산 그래프(Computational Graph):** 복잡한 미분 수식을 노드 단위의 국소적 계산으로 분해하여 연산 효율을 높이는 과정 습득.
- **구현 경험:** `MulLayer`, `AddLayer` 등 각 계층을 클래스로 모듈화하여 순전파와 역전파를 직접 구현함.

### 3. CNN (Convolutional Neural Networks, Ch 7) 🖼️
- **영상 처리와의 연결:** `Digital Image Processing` 수업에서 배운 **Spatial Filtering** 개념을 딥러닝의 특징 추출 과정과 결합하여 이해함.
- **Im2Col 최적화:** 고속 연산을 위해 4차원 데이터를 2차원 행렬로 전개하여 처리하는 기법 학습. (HW 가속기 설계 시의 연산 최적화 관점에서 흥미로웠음)
